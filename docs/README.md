## 날짜를 입력받는다.

- 숫자가 아닐 계우 예외처리
- 1이상 31이하 숫자가 아닐 경우 예외처리
- 예외 처리 후 다시 입력받기

## 메뉴를 입력받는다.

- 메뉴판에 없는 메뉴 예외처리
- 메뉴 갯수 1이상 숫자외에 예외처리
- 메뉴-“갯수” 형식이 다를 경우 예외처리
- 중복메뉴 입력시 예외처리 -메뉴는 한번에 20개까지 가능
- 예외처리 후 다시 입력받기

## 할인 전 총 금액 계산 후 저장

- 총 금액이 10,000이하 이벤트 적용 제외
- 음료만 주문시 이벤트 적용 제외

## 크리스마스 디데이 할인 날짜인지 확인 (12.1 - 12.25)

- 입력받은 날짜가 12.1~12.25인지 확인
- 할인 금액 계산 1일부터 1,000원시작 100원씩 증가

## 평일인지 확인 (일~목)

-입력받은 날짜가 평일인지 확인 -입력받은 메뉴 중 디저트 메뉴 1개당 2,023원할인

## 주말인지 확인(금,토)

-입력받은 날짜가 금,토인지 확인 -입력받은 메뉴 중 메인 메뉴 1개당 2,023원 할인

## 별이 있는 날짜인지 확인

- 입력받은 날짜가 (3일,10일,17일,24일,25일,31일)
- 총 주문에서 -1,000원 할인

## 총 금액이 12만원 이상인지 확인

- 샴폐인 1개 증정(25,000원 상당)

## 혜택 금액에 따른 12월 이벤트 배지 부여 후, 저장

- 5천 원 이상: 별
- 1만 원 이상: 트리
- 2만 원 이상: 산타

## 주문메뉴를 출력한다.

## 할인 전 주문금액을 출력한다.

## 증정메뉴를 출력한다.

- 해당안될 경우, “없음” 출력

## 혜택내역을 출력한다.

- 혜택 내용 없을 경우, “없음”출력

## 총 혜택금액을 출력한다. (= 할인 금액의 합계 + 증정 메뉴의 가격)

## 할인 후 예상 결제금액을 출력한다. (= 할인 전 총주문 금액 - 할인 금액)

## 12월 이벤트 배지를 출력한다.

- 해당안될 경우, "없음" 출력
